# 🔨 KorupSmash

KorupSmash adalah game web sederhana bertema **pukul tikus (whack-a-mole)** di mana pemain harus menekan target yang muncul untuk mendapatkan skor dalam batas waktu tertentu.

---

## 🌐 Live Demo

👉 [https://anandadavidsaputra.github.io/KorupSmash/](https://anandadavidsaputra.github.io/KorupSmash/)

---

## 🧩 Overview

KorupSmash dibangun sebagai proyek latihan pengembangan web untuk:

* menampilkan target (tikus) secara acak di layar
* menghitung skor berdasarkan jumlah pukulan yang berhasil
* menerapkan sistem countdown timer
* menampilkan hasil akhir berupa status/kelas berdasarkan total skor

Game ini menguji kecepatan dan respons pemain dalam waktu yang terbatas, sekaligus melatih implementasi logika permainan menggunakan JavaScript.
